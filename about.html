About Sistem Smart Home

Sistem Smart Home ini merupakan aplikasi berbasis web yang digunakan untuk mengontrol perangkat elektronik rumah seperti lampu dapur, kamar, dan ruang tamu secara jarak jauh.

Aplikasi ini terhubung langsung dengan perangkat IoT (ESP32) melalui jaringan lokal menggunakan metode HTTP request. Setiap perintah yang dikirimkan akan langsung mengubah status perangkat secara real-time.

Platform: Web-Based

Perangkat IoT: ESP32

Bahasa: HTML, CSS, JavaScript

Fungsi Utama: Kontrol lampu ON / OFF

Sistem ini dikembangkan sebagai bagian dari proyek pembelajaran Internet of Things (IoT) dengan fokus pada integrasi hardware dan antarmuka web yang sederhana, stabil, dan mudah digunakan.
